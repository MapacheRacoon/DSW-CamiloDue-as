Sintaxis y Selectores CSS

Un conjunto de reglas CSS consta de un selector y un bloque de declaración

h1:es el selector {color: es la propiedad, blue: es el valor y esto es conjunto es la declaracion}

Los aspectos selector para el componente HTML que quiere estilo. El bloque de testimonio tiene una o más declaraciones separadas por punto y coma. Una aseveración CSS constantemente acaba con un punto y coma, y ​​los bloques de declaraciones permanecen rodeados por llaves.

Los selectores CSS

Selectores CSS se usan para "find" (o seleccionar) recursos HTML en funcionalidad de su nombre de factor, de la identificación, clase, atributo, y muchísimo más.

El selector de componente

Puede elegir todos los recursos en una página como esta (en esta situación, todos los recursos van a ser alineados-centro, con un color rojo de texto)

El selector de ID

Este usa el atributo id de un componente HTML para elegir un componente específico. La identificación de un factor debería ser exclusivo en una página, por lo cual el selector de ID se usa para elegir un factor único!

El selector de clase

selecciona los recursos con un atributo de clase específica. Para elegir recursos con una clase específica, redacta un Carácter, seguido del nombre de la clase.

CSS Comentarios

Los comentarios se aplican para describir el código, y tienen la posibilidad de auxiliar al editar el código fuente en una fecha siguiente. Los comentarios son ignorados por los navegadores.
